# Customer Churn Prediction (MLOps Project)

This is an MLOps project that predicts whether customers will churn (leave the service) or stay.

## 🚀 Live Demo
You can check out the live application here:
[https://mlops-churn-prediction-nl3tyhn4mjuzgf57ydoxjy.streamlit.app/](https://mlops-churn-prediction-nl3tyhn4mjuzgf57ydoxjy.streamlit.app/)

## 🛠 Technologies Used
- **Python**
- **Streamlit** (for UI)
- **Scikit-learn** (for modeling)
- **Git & GitHub** (for version control)

## 📂 Project Structure
- `src/`: Contains core scripts (`app.py`, `predict.py`, `train.py`)
- `requirements.txt`: Project dependencies
